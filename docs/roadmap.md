## 3. Вторая итерация (после первого прода)

### 3.1 Фото пирогов
- Клиент → JPG/WebP → `frontend-next/public/menu/`
- Заполнить `image: '/menu/...'` в `lib/menu.ts` (62 позиции, но актуальны топ 20-30)

### 3.2 Настоящий backend (auth + заказы)
- Выбрать VPS (Timeweb/Selectel/Beget, ~300 ₽/мес)
- Развернуть `backend/` — готов: Express + SQLite + nodemailer + DaData
- Фронт: `NEXT_PUBLIC_API_BASE` вместо Web3Forms
- Работающий личный кабинет, история заказов

### 3.3 DaData — автодополнение адреса
- API-ключ от клиента
- Подключить к `AddressForm.tsx`
- Валидация зоны доставки (Москва, в пределах МКАД)

### 3.4 Промокоды / скидки
- Бизнес-правила от клиента
- Реализация в бэкенде
- Активация UI-поля

### 3.5 Эквайринг
- Провайдер: ЮKassa / Tinkoff / CloudPayments
- Договор с банком — клиент, 1-2 недели
- Интеграция
