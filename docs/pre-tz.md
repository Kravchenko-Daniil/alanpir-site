## Развилки — решить с Даниилом (Рамиль не нужен)

1. **Прайс/веса.** Прайс Рамиля местами расходится с меню и по ценам, и по весам
   (напр. осетинский Фыдджын: сейчас 700/1000/1200 за 1020/1340/1450, у Рамиля 600/1000/1200 за 990/1290/1390).
   Решить: прайс Рамиля = источник истины (перезаписать цены+веса)? Что с позициями вне его списка (соусы, напитки)?
2. **Зоны доставки.** Рамиль дал зоны с разными минималками/ценой:
   соседние станции (Достоевская/Марьина роща) — бесплатно; Садовое — мин 1000 / 150 ₽;
   ТТК — мин 1500 / 250 ₽; МКАД — мин 2000 / 250 ₽; за МКАД — от 5000, по согласованию.
   Сейчас в коде: единый мин 2000, free от 5000, flat 140 ₽.
   Решить: текстом сейчас + расчёт позже / ручной выбор зоны в чекауте / автоопределение по адресу (DaData).
3. **Промокоды.** У Рамиля промокодов нет — только акции-подарки. Поле «Промокод» в чекауте: убрать или оставить заделом?
4. **Лого.** Фото девушки в костюме на красно-жёлтом фоне — не вектор, конфликтует с оранжевым брендом (`#F25C05`).
   Решить: использовать как есть / только на отдельных блоках / заказать перерисовку / пока не трогать.
5. **«Пирог в кадр» (доп. просьба).** Фото обрезаются (`object-cover`). Переключить на показ целиком
