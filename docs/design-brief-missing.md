---

Дополни существующий Next.js 15 + React 19 + Tailwind 4 проект «АланПир» (осетинские пироги, Москва). Сохрани текущий дизайн-язык без изменений. Сделай недостающие страницы и переиспользуемые компоненты. Логику не реализуй — только UI с моками через `useState`, заглушки функций.

## Дизайн-токены (обязательно использовать как есть)

- Цвета: `--color-terracotta: #C04D29`, `--color-bg-warm: #FAF9F6`, `--color-ink: #1D1D1B`, `--color-muted: #6B6B6B`, `--color-surface: #FFFFFF`, `--color-border-warm: #E5E1DA`
- Шрифты: `Inter` (UI, дефолт), `Playfair Display` (через `.font-serif` / `var(--font-serif)` для заголовков)
- Скругления: `rounded-xl` / `rounded-2xl` для блоков, `rounded-full` для пилюль
- Карточки: `bg-surface rounded-2xl border border-border-warm`, без теней
- Лейаут: `max-w-[940px] mx-auto px-4 sm:px-6 lg:px-8`
- Иконки: **только `lucide-react`**, никаких эмодзи в иконках
- Анимации: `motion` (framer-motion)
- Стек: без новых npm-пакетов, только уже установленные (`motion`, `lucide-react`, `clsx`, `tailwind-merge`)

Используй те же паттерны, что уже в проекте (стили формы из checkout, StepCard, OrderSummary).

## Что важно

- **Промокоды — заглушка** на этом этапе. Показывать UI поля промокода, но с припиской «Промокоды временно недоступны» или кнопку «Применить» задизейблить. Логику скидок за промокод реализовывать **не нужно**.
- **Скидки — тоже заглушка**: не считай реальные проценты, просто показывай 0 ₽ в строке «Скидка» (или скрывай её, если 0).
- Все компоненты — `'use client'` (Next.js 15 client components).
- Никакой реальной логики отправки/запросов — все коллбэки `() => {}` или `console.log(...)`.
- Состояние модалок/корзины — через локальный `useState` в layout или простой контекст, без Zustand/Redux.
- Код на TypeScript, компоненты — дефолтный экспорт.

## Формат ответа

Отдай **каждый файл целиком**, без фрагментов и многоточий. Структура:

```
app/
  order/
    [id]/
      page.tsx
  account/

  page.tsx                 # обновлённый (hero + акции)

components/
  CartDrawer.tsx

hooks/
  useToast.ts
```
