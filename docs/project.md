## В каком состоянии проект

Готов на ~80%. Полный флоу работает: регистрация → корзина → чекаут с подсказками адреса → заказ сохраняется в БД → история в личном кабинете.

**Главное узкое место — Рамиль.** Клиент должен прислать прайс, фото, тексты, логотип, баннеры и доступы. Без этого финальный запуск не сдвинется. Технически — почти всё готово.

## Стек

- **Фронт**: Next.js 15 + React 19 + Tailwind 4 + TypeScript. Статический экспорт (`output: 'export'`).
- **Бэк**: Node.js + Express + SQLite + nodemailer + DaData-прокси.
- **Дизайн**: оранжевая палитра `#F25C05`, шрифты Manrope + Fraunces, контейнер 1360px.
- **Меню**: 62 позиции в `frontend-next/lib/menu.ts`.

## Жёсткие ограничения (не нарушать)

- REG.RU — shared-хостинг, **Node.js там не работает**. API только на внешнем VPS.
- В `next.config.ts` обязательно `output: 'export'` + `images.unoptimized: true`.
- Динамические роуты `[id]` несовместимы со статическим экспортом — используем query-параметры (`/order?id=…`).
- В `globals.css` оставлять legacy-alias `--color-terracotta` = `--color-accent` — иначе чекаут/кабинет/модалки сыпятся.
- Секреты (`.env`, `*.sqlite`, `secrets-prod`) в git не коммитим.
- Email из учётки Claude Code (`krawchenkodaniil@gmail.com`) **не использовать** в проекте — это рабочая учётка, не клиентский email.
