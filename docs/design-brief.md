# Промпт для Gemini — UI/UX для сайта АланПир

> Скопировать всё ниже одним сообщением в Gemini.

---

Ты — senior product-дизайнер. Спроектируй современный UI/UX для сайта осетинской пекарни-доставки «АланПир» (Москва). Дай: макеты всех страниц (описание секций и их расположения), компонентную систему, интерактивные флоу и mobile-first адаптивные раскладки. Формат ответа — текстовый дизайн-док с чёткой структурой, без кода.

## Бренд и контекст

- Название: **АланПир** (alanpir.ru)
- Что продают: осетинские пироги (мясные, сырные, с зеленью, рыбные, сладкие), сеты, закуски, напитки
- География: Москва, самовывоз и доставка
- Аудитория: семьи, офисы (корп-заказы), любители кавказской кухни, возраст 25–55
- Текущий тон-оф-войс: тёплый, домашний, традиционный, но без «деревенского китча»
- Ключевое преимущество: горячие пироги за 20 минут, натуральные ингредиенты, ручная лепка

## Дизайн-требования (минимум)

- **Светлая тема**, один акцентный цвет (предложи на выбор тёплые осетинские мотивы: терракот, охра, тёмно-красный)
- **Минимализм**: много воздуха, сетка 12 колонок, крупная типографика, мало текста — главное сразу видно
- Геометрия — мягкие скругления, без теней-пузырей
- Фото пирогов — главный визуальный элемент (крупные, аппетитные, на чистом фоне)
- Шрифт: современный гротеск для UI, допустим акцентный шрифт в логотипе/заголовках
- Адаптивность: mobile-first, нормально работать на экранах 360–1440 px
- Доступность: контраст AA, фокус-стили, клавиатурная навигация

## Страницы (ровно такая структура, со всей функциональностью)

### 1. Главная (`/`)
- Верхняя шапка с навигацией (см. «Навигация»)
- Hero-блок: слоган «Доставка горячих осетинских пирогов по Москве от 20 минут», главный CTA «Смотреть меню», фоновое фото пирога
- Три баннера-преимущества: «Традиционные рецепты», «Натуральные ингредиенты», «Рейтинг 5★»
- Секция «Хиты»: 3–4 карточки топовых пирогов с фото, названием, ценой, кнопкой «В корзину»
- Секция «Акции»: карточки текущих акций:
  - 10% при самовывозе
  - Бесплатная доставка от 5000 ₽
  - Фирменный нож в подарок от 7000 ₽
  - Пирог в подарок от 3000 ₽
- Футер (см. «Футер»)

### 2. Каталог (`/catalog`)
- Page-hero с заголовком «Все пироги и сеты»
- Панель фильтров (sticky при скролле): табы по категориям — Все / Новинки / Хиты / Мясные / Сырные / Постные / Десерты / Закуски
- Тогл «Показать только постные»
- Поле ввода промокода с кнопкой «Применить»
- Сетка карточек товаров (2-3-4 колонки в зависимости от ширины):
  - Фото, бейджи (новинка/хит/акция), название, вес (варианты 600/1000/1200 г), цена, кнопка «В корзину»
- Клик по карточке — модалка товара с подробным описанием, селектором размера/веса, количества, кнопкой «Добавить в корзину»

### 3. О нас (`/about`)
- Page-hero «О пекарне АланПир»
- Текстовый блок с историей (2–3 абзаца, коротко)
- Сетка из 4 карточек-преимуществ с иконкой/фото:
  - Лучшее мясо — фермерское, сочная начинка
  - Натуральные сыры — осетинские, тянущаяся начинка
  - Наши пекари — ручная лепка, печём перед доставкой
  - Отборная мука — тонкое тесто
- Блок с фото производства / команды

### 4. Отзывы (`/reviews`)
- Page-hero «Что говорят наши гости»
- Сетка карточек отзывов (имя, дата, звёзды, текст) — заглушки на 6–9 штук
- CTA-ссылка «Читать все отзывы на Яндекс Картах» (внешняя)
- Форма «Оставить отзыв» (имя, оценка звёздами, текст)

### 5. Личный кабинет (`/account`)
Два состояния:
- **Не авторизован**: крупная карточка «Войдите, чтобы видеть свои заказы и адреса» + кнопка «Войти/Регистрация»
- **Авторизован**:
  - Приветствие «Здравствуйте, {имя}»
  - Три info-карточки: мои данные (телефон, email), мои адреса, мои заказы
  - Список заказов с датой, статусом, суммой, кнопкой «Повторить заказ»
  - Кнопка «Выйти»

### 6. Оформление заказа (`/checkout`)
Пошаговый флоу в одной странице (или 3 шага с прогресс-баром):
1. **Контакты**: имя, телефон, email (если не авторизован)
2. **Доставка**:
   - Переключатель: Доставка / Самовывоз
   - Для доставки: инпут адреса с автоподсказками (интеграция DaData — регион → город → улица → дом → квартира)
   - Сохранённые адреса (если авторизован) — выбор из списка + «Добавить новый»
   - Дата и время доставки (слоты 08:30–19:00)
   - Комментарий к заказу
3. **Оплата**: карта (МИР/Visa/MC) или СБП
- Справа (на десктопе) — sticky-сайдбар с составом корзины, скидками, итогом
- Поле промокода
- Финальная кнопка «Оформить заказ»

### 7. Спасибо-страница (`/order`)
- Большая иконка успеха
- Заголовок «Заказ №XXXX принят»
- Карточка с деталями: список товаров, адрес, время, сумма
- Кнопки «В каталог» и «В личный кабинет»
