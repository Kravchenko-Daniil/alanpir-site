## Дизайн-направление

### Палитра

**Подбери палитру сам** под бренд «осетинские пироги». Главное — **тёплая, аппетитная, коммерческая** (доставка еды, а не премиум-минимализм). Избегай слишком насыщенного жёлтого, поскольку это визуальный маркер конкурентов.

Возможные направления:
- Терракота + кремовый + глубокий тёмный (восточный, «кавказский»)
- Тёплый охристый + молочный + уголь
- Бордовый + слоновая кость + золотистые акценты (премиум, но тёплый)
- Тёмно-зелёный + песочный + коричневый (эко, домашний)

Предложи **один** вариант и объясни в одном абзаце, почему он подходит бренду «АланПир» (осетинские традиции, ручная лепка, тёплая еда).

### Шрифты

Текущие: `Inter` (UI) + `Playfair Display` (заголовки). Можешь оставить, можешь предложить что-то более «фудовое» (например, `Rubik` + `Roboto Slab`, или `Manrope` + `Fraunces`). Все шрифты должны поддерживать кириллицу.

### Скругления / стиль

- Контейнер `max-w-[1360px]` сохранить (уже устраивает клиента)
- Карточки: `rounded-2xl`, лёгкий бордер, **без тяжёлых теней** (легкая тень при hover — ок)
- Кнопки: `rounded-full` для второстепенных, `rounded-xl` для CTA

### 4. `app/globals.css` — токены

Сохрани Tailwind 4 `@theme {}` структуру, обнови цвета. Пример:

```css
@import "tailwindcss";

@theme {
  --color-accent: #...;       /* главный акцент (жёлтый или что выберешь) */
  --color-accent-dark: #...;  /* hover */
  --color-ink: #111;
  --color-muted: #6B6B6B;
  --color-bg: #FFFFFF;
  --color-bg-warm: #FAF9F6;
  --color-surface: #FFFFFF;
  --color-border-warm: #EAEAEA;
  --color-danger: #E0433A;
  --color-terracotta: var(--color-accent); /* legacy alias — чтобы не сломать checkout/account */

  --font-sans: var(--font-inter), ui-sans-serif, sans-serif;
  --font-serif: var(--font-playfair), ui-serif, Georgia, serif;
}
```

Важно: **обязательно оставить alias `--color-terracotta`** (приравнять к акценту) — иначе существующие компоненты checkout/account/модалки посыпятся.
