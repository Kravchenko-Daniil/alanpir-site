# Промпт для Gemini — редизайн главной и шапки АланПир

## Задача

Редизайн главной страницы и шапки сайта-доставки осетинских пирогов «АланПир». Проект на Next.js 15, он уже живёт — работают каталог, корзина, checkout, модалки. Их не трогаем.

## Что должно получиться

Современный тёплый коммерческий дизайн (не премиум-минимализм). Запоминающийся собственный бренд. Никакого копирования чужих сайтов.

## Результат

Четыре файла целиком:
- `app/globals.css` — обновлённые дизайн-токены (палитра, шрифты)
- `components/Header.tsx` — шапка с телефоном, часами, навигацией, корзиной
- `components/Footer.tsx` — подвал с контактами, реквизитами, разделами
- `app/page.tsx` — главная: баннер-слот, заголовок, преимущества, секции по категориям (Хиты / Осетинские / Мясные / Сладкие), блок акций

## Обязательные ограничения

- Брать продукты только из `@/lib/menu` (готовый источник, 62 позиции)
- Использовать хуки `useCart`, `useAuthModal`, `useProductModal` (готовые)
- Компонент `<ProductCard product={p} />` — готов, использовать его
- Оставить в `globals.css` legacy-alias `--color-terracotta` (равным акценту), иначе сломаются другие страницы
- `output: 'export'` — только client-side, никаких server actions
- Не использовать жёлтую шапку и лого в квадратной чёрной рамке (ассоциация с конкурентом)
- Не копировать тексты и названия продуктов с других сайтов
